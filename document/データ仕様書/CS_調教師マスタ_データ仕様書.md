# JRDB 調教師データ仕様書

> 第１版　最終更新：2001.05.16

---

## ファイル概要

| 項目 | 内容 |
|------|------|
| **ファイル名** | `CZA` + 年月日 + `.txt`（全調教師）／`CSA` + 年月日 + `.txt`（今週出走分・先週成績更新分）　例）`CZA010504.txt`（2001年5月4日） |
| **圧縮形式** | `.lzh`（lha圧縮） |
| **更新日時** | 木曜日（想定確定後）19:00頃 |
| **ファイル形式** | 固定長テキストファイル |
| **レコード長** | 272 BYTE |

---

## 型定義

| TYPE | 説明 |
|------|------|
| `9` | 数字（0のとき0） |
| `Z` | 数字（0のとき空白） |
| `X` | 文字 |

- `OCC`：繰り返し定義
- `BYTE`：項目長
- 数字は `"-"` マイナス記号有り

## 注記

- **データ範囲**：1996年度以降、ＪＲＡのレースに出走した馬の調教師。
- **引退調教師**：ＪＲＡ所属以外の調教師については、登録抹消フラグ・登録抹消年月日は入りません。
- **リーディング**：ＪＲＡ所属以外の調教師はリーディングの対象外。関東・関西別で計算。
- **成績**：成績の無い場合はスペース。
