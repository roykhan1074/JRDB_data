# JRDB 馬基本データ仕様書

> 第３版　最終更新：2002.08.20

---

## 概要

競走馬に関する基本的なデータです。競走馬データの補完として使用します。

---

## ファイル概要

| 項目 | 内容 |
|------|------|
| **ファイル名** | `UKC` + 年月日 + `.txt`　例）`UKC020713.txt`（2002年7月13日） |
| **圧縮形式** | `.lzh`（lha圧縮） |
| **更新日時** | 金曜日 19:00頃 / 土曜日 20:00頃 |
| **ファイル形式** | 固定長テキストファイル |
| **レコード長** | 292 BYTE |

---

## 型定義

| TYPE | 説明 |
|------|------|
| `9` | 数字（0のとき0） |
| `Z` | 数字（0のとき空白） |
| `X` | 文字 |

- `OCC`：繰り返し定義
- `BYTE`：項目長

## 注記
