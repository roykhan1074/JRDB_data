# JRDB 成績データ速報版仕様書

> 第４版　最終更新：2007.10.22

---

## 概要

成績データ（SEC）の正式版は木曜日に提供しますが、レース当日で提供可能なデータを速報版として提供します。

---

## ファイル概要

| 項目 | 内容 |
|------|------|
| **ファイル名** | 成績データ（SEC）と同一。`SEC` + 年月日 + `.txt`　例）`SEC000105.txt`（2000年1月5日） |
| **圧縮形式** | `.lzh`（lha圧縮） |
| **更新日時** | レース当日 18:00頃（木曜日に正式版「成績データ(SEC)」によって上書き） |
| **ファイル形式** | 固定長テキストファイル |

---

## 型定義

| TYPE | 説明 |
|------|------|
| `9` | 数字（0のとき0） |
| `Z` | 数字（0のとき空白） |
| `X` | 文字 |
| `F` | 16進数（数字 or 小文字アルファベット） |

- `OCC`：繰り返し定義
- `BYTE`：項目長
- 数字は `"-"` マイナス記号有り

---

## その他

データフォーマットは成績データと同一です。成績データが改版された場合、この成績データ速報版も自動的に改版されます。

---
