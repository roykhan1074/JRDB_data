# JRDB 競走馬データ仕様書

> 第11版　最終更新：2023.06.25

---

## ファイル概要

| 項目 | 内容 |
|------|------|
| **ファイル名** | `KYI` + 年月日 + `.txt`　例）`KYI080913.txt`（2008年9月13日） |
| **圧縮形式** | `.lzh`（lha圧縮） / `.zip`（zip圧縮・推奨版） |
| **更新日時** | 金曜日 19:00頃 / 土曜日 20:00頃 |
| **ファイル形式** | 固定長テキストファイル |
| **レコード長** | 1024 BYTE |

---

## 型定義

| TYPE | 説明 |
|------|------|
| `9` | 数字（0のとき0） |
| `Z` | 数字（0のとき空白） |
| `X` | 文字 |
| `F` | 16進数（数字 or 小文字アルファベット） |

- `OCC`：繰り返し定義
- `BYTE`：項目長
- 数字は `"-"` マイナス記号有り

各項目の内容は「競走馬データ仕様書の説明」および「ＪＲＤＢデータコード表」を参照。
