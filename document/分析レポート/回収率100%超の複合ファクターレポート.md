## 第4章：パターン構造の分析

### 全100%超パターンに共通する要素

```
必須要素（ほぼ全パターンに存在）:
  ├── 情報印1位（またはLS指数2〜3位）
  └── 前週追切指数60以上（週をまたいだ調教好調の継続）

強化要素（追加で加えると効果が上がる）:
  ├── 仕上指数70以上
  ├── 放牧先ランクA/B
  ├── 騎手期待連対率20%以上
  └── 上がり指数1位

市場条件（オッズ帯が重要）:
  └── 4〜15倍（本命外・中穴ゾーン）が最も効果的
      ← 1〜3倍は過剰人気で回収率が下がる
      ← 15倍以上はサンプルが少なく不安定
```

### なぜ4〜15倍が最も有効か

```
高能力×高コンディション馬が4〜15倍になる理由：
  1. 前走着外（人気落ち）
  2. 騎手が一般的に有名でない
  3. 馬主・厩舎が地味
  4. 前走間隔が長い（放牧明け）

→ JRDBの指数・前週調教は正確に評価しているが
  一般馬券購入者がその情報にアクセスしていない
```

### 前週追切指数が重要な理由

前週（1週前）の追切指数が「70以上」ということは：
- 最終追切（当週）だけでなく、**週をまたいで好調を維持**
- 疲れが抜けた状態で最終調整に臨んでいる証拠
- 単なる「最終追切が良い」より**信頼性が高い**

## 第5章：実践的な活用法

### チェックリスト（買い条件）

```
□ 情報印が1位（◎）
□ 前週追切指数が60以上（70以上なら確度UP）
□ 基準オッズが4〜15倍
□ 以下のうち2つ以上を満たす:
   □ 仕上指数70以上
   □ 放牧先ランクA/B  
   □ 騎手期待連対率20%以上
   □ 厩舎指数10以上
   □ 上がり指数順位1位
```

### リスク管理

| 条件数 | 単回収目安 | 1レースあたりの出現率 | 推奨ベット |
|---|---|---|---|
| 3条件（情報1+前週追切+オッズ） | 〜103 | 比較的多い | 普通 |
| 4条件 | 〜110 | 少ない | やや多め |
| 5条件 | 〜126 | 非常に少ない | 小額（n不足） |

### 注意事項

1. **前週追切指数はT_CYBのisshuumae_oi_indexフィールド**から取得。データがないレースは対象外
2. **全条件が揃う馬は1開催で数頭〜十数頭**程度と少ない
3. n<200のパターンは2026年以降も継続してデータを積み上げて検証が必要
4. 特定の年度・季節・コースへの偏りがある可能性がある（全コース・全期間の合算のため）
5. **単回収100%超でも複勝回収率が90%台のものは「勝ちか負けか」の2択**になりやすい。複勝率90%以上のパターンと組み合わせて使用することを推奨
