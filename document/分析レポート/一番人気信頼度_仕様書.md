## 概要

**1番人気馬をJRDB指数群が客観的に支持しているかを S〜E の6段階で評価する指数。**

- オッズ（市場評価）は**入力に使わない**（市場評価から独立した指標）
- 低グレード（D/E）＝「市場は支持しているが、データ的な裏付けが薄い」＝偽本命シグナル
- 高グレード（S/A）＝「市場と複数のJRDB指数が一致」＝信頼できる本命

---

## グレード定義とバックテスト結果

| グレード | スコア閾値 | 実績勝率 | 実績連対率 | 実績複勝率 | レース数 | 意味 |
|---|---|---|---|---|---|---|
| S | ≥ 70 | **43.4%** | 64.3% | 75.9% | 4,142 | 全指数が強く支持。迷わず本命視できる |
| A | 55〜69 | **36.0%** | 56.0% | 68.7% | 3,102 | 信頼できる本命 |
| B | 25〜54 | **30.3%** | 50.4% | 63.5% | 7,288 | 標準的な1番人気（全体平均31.4%に一致） |
| C | 0〜24 | **24.3%** | 42.3% | 54.7% | 4,645 | 平均以下。要注意 |
| D | −20〜−1 | **19.4%** | 33.7% | 46.6% | 1,426 | 支持が薄い。危険信号 |
| E | < −20 | **13.0%** | 27.9% | 37.5% | 323 | 複数の赤フラグ。偽本命の可能性大 |

> 全体の1番人気勝率：31.4%（20,926レース）

BグレードがちょうどD/Eの「平均以下」と、S/Aの「平均以上」の境界になるよう閾値を設計している。

## スコアリング詳細

対象馬：`kijun_ninki = 1`（基準人気1位の馬）のみ  
計算方式：各ファクターのレース内相対ランクに基づくポイント加算

### A. IDM優位性（最大+40 / 最小−20）

`idm`（JRDB個別馬評価値）のレース内順位と、2位との差分を評価。

| 条件 | 点数 |
|---|---|
| IDM1位 かつ 2位との差 ≥ 5pt | +40 |
| IDM1位 かつ 差 3〜4pt | +30 |
| IDM1位 かつ 差 1〜2pt | +20 |
| IDM1位 かつ 差 0（同値） | +10 |
| IDM2位 | 0 |
| IDM3位 | −10 |
| IDM4位以下 | −20 |

### B. 情報指数一致（最大+20 / 最小−15）

`joho_index`（JRDB情報指数）のレース内順位を評価。  
`joho_index = −1`（非開示）の馬は集計から除外し、ファクター全体を中立（0点）扱い。

| 条件 | 点数 |
|---|---|
| joho_index 1位 | +20 |
| 2位 | +5 |
| 3位 | 0 |
| 4〜5位 | −5 |
| 6位以下 | −15 |

### C. 展開有利性（最大+15 / 最小−10）

`goal_juni`（JRDB展開指数）のレース内順位を評価。値が小さいほど展開有利（昇順ランク）。

| 条件 | 点数 |
|---|---|
| 1位（最有利） | +15 |
| 2〜3位 | +10 |
| 4〜5位 | +5 |
| 6〜7位 | 0 |
| 8位以下 | −10 |

### D. 調教矢印（最大+15 / 最小−20）

`chokyo_yajirushi`（調教方向評価）を直接スコア化。

| 矢印コード | 表示 | 点数 |
|---|---|---|
| 1 | ↑↑（急上昇） | +15 |
| 2 | ↑（上昇） | +8 |
| 3 | →（横ばい） | 0 |
| 4 | ↓（下降） | −10 |
| 5 | ↓↓（急下降） | −20 |

### E. 追切・仕上指数（各最大+8 / 最小−5）

`oi_index`（追切指数）、`shiage_index`（仕上指数）をそれぞれレース内順位で評価。

| 順位 | 点数 |
|---|---|
| 1位 | +8 |
| 2〜3位 | +3 |
| 4〜5位 | 0 |
| 6位以下 | −5 |

### F. ペース適性（合計最大+10 / 最小−6）

`ten_index_juni`（テン指数順位）と `agari_index_juni`（上がり指数順位）を各々評価して合算。  
※ これらは T_KYI に既にランク値として格納されているため、追加集計不要。

| 順位 | 点数（1指標あたり） |
|---|---|
| 1位 | +5 |
| 2〜3位 | +2 |
| 4〜6位 | 0 |
| 7位以下 | −3 |

## スコア合計とグレード変換

```
合計スコア = A + B + C + D + E（追切）+ E（仕上）+ F（テン）+ F（上がり）

S : score ≥ 70
A : 55 ≤ score < 70
B : 25 ≤ score < 55
C :  0 ≤ score < 25
D : −20 ≤ score < 0
E : score < −20
```

## 実装メモ

- 計算はフロントエンド（JavaScript）で実施。バックエンド・DB変更不要
- 全馬データ（`/api/races/:key/entries` のレスポンス）を受け取った後に `renderTable()` 呼び出し前に実行
- `joho_index` が全馬 −1（非開示）の場合、Bファクターは全馬0点となり事実上除外される
- IDMが0または欠損の場合、Aファクターはスキップ（0点）
