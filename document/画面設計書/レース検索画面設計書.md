## 2. 検索フォーム

| 項目 | 種別 | 初期値 | 送信パラメータ |
|---|---|---|---|
| 日付 | `<input type="date">` | 当日 | `date=YYYYMMDD` |
| 競馬場 | `<select>` | 全場 | `course=01〜10`（全場のとき省略） |

| 値 | 表示 |
|---|---|

| `01` | 札幌 |

| `04` | 新潟 |
| `05` | 東京 |
| `06` | 中山 |
| `07` | 中京 |
| `08` | 京都 |
| `09` | 阪神 |
| `10` | 小倉 |

## 3. 検索結果

結果は競馬場ごとにグループ分けして表示。各グループ内はレース番号順。

### 一覧カラム

| 列 | 内容 | 備考 |
|---|---|---|
| R | レース番号 | 整数表示 |
| 発走 | 発走時刻 | `start_time`（HHMM→H:MM形式） |
| レース名 | レース名略称（9文字）または正式名 | 重賞はグレードバッジ付き |
| 距離 | 芝/ダバッジ + 距離（m） | |
| 頭数 | 登録頭数 | |
| ボタン | 出馬表を見る → `/entries.html?key=XXXXXXXX` | 常時「出馬表を見る」（青ボタン） |

### グレードバッジ

| grade値 | 表示 |
|---|---|
| `1` | G1 |
| `2` | G2 |
| `3` | G3 |
| `A` | L |

### 芝/ダバッジ

| tds_code | 表示 | バッジ色 |
|---|---|---|
| `1` | 芝 | 緑（#dcfce7） |
| `2` | ダ | オレンジ（#ffedd5） |
| `3` | 障 | 紫（#ede9fe） |

## 4. API仕様

### GET /api/races?date=YYYYMMDD[&course=XX]

**レスポンス（JSON配列）**

```json
[
  {
    "course_code": "08",
    "year_code":   "26",
    "kai":         "3",
    "day_code":    "2",
    "race_num":    "10",
    "race_name":   "センテニアル・ＰＳ",
    "race_name_9char": "センテニアルＰＳ",
    "distance":    "1800",
    "tds_code":    "1",
    "grade":       "5",
    "heads":       "18",
    "start_time":  "1450",
    "data_kubun":  "3"
  }
]
```

**ソート：** course_code ASC, race_num（数値）ASC

## 5. 画面遷移

| 遷移先 | 条件 |
|---|---|
| `/entries.html?key=XXXXXXXX` | 「出馬表を見る」ボタンクリック |
| `/` | ヘッダー「← ポータルに戻る」クリック |

### レースキー（8桁）の構造

```
course_code(2) + year_code(2) + kai(1) + day_code(1) + race_num(2)
例: 08 26 3 2 10 → "08263210"
```
