## 3. レースカードヘッダー

| 表示要素 | データソース | 備考 |
|---|---|---|
| レース番号 | `race.race_num` | 整数表示 |
| レース名 | `race.race_name_9char` or `race.race_name` | 9文字略称優先 |
| クラスバッジ | `race.grade`（G1/G2/G3/L）+ `race.class`（3勝/OP等） | gradeが優先、なければclassで判定 |
| 開催日 | `race.ymd` | YYYY/MM/DD形式 |
| 競馬場 | `race.course_code` → COURSE_NAME | |
| 発走時刻 | `race.start_time` | HHMM → H:MM |
| 芝/ダバッジ | `race.tds_code` | |
| 左右 | `race.migihidari` | 1:右 2:左 |
| 距離 | `race.distance` + "m" | |
| 頭数 | `race.heads` + "頭" | |

### クラスバッジ対応表

| race.grade | 表示 |

### カラム一覧

| 列 | DBカラム | 表示ルール |
|---|---|---|

| 馬名 | `uma_name` | |

| 脚質 | `kyakushitsu` | 1:逃げ/2:先行/3:差し/4:追込 |
| オッズ | `kijun_odds` | 1位=ピンク背景、2〜3位=緑背景 |
| 騎手 | `kishu_name` | |
| 厩舎 | `trainer_name` | |
| 情報印 | 情報指数の Dense Rank | ◎○▲注△▽（値=-1のとき非表示） |
| 情報指数 | `joho_index` | 値=-1のとき薄グレー背景 |
| 展開印 | ゴール順位の Dense Rank | ◎○▲注△▽ |
| 展開指数 | `goal_juni` | |
| IDM印 | IDMの Dense Rank | ◎○▲注△▽ |
| IDM指数 | `idm` | IDM3位以下5pt差→グレー背景 |
| 前3F順位 | `ten_index_juni` | 1位=ピンク/2〜3位=緑 |

| 仕上指数 | `shiage_index`（T_CYB） | ≥60=緑/＜40=赤 |
| 調教スコア | オリジナル計算値 | S/A/B/C/D + 数値 |

### 印の順位→記号マッピング

| 順位 | 記号 |
|---|---|
| 1 | ◎ |
| 2 | ○ |
| 3 | ▲ |
| 4 | 注 |
| 5 | △ |
| 6 | ▽ |
| 7以上 | — |

印はレース内の Dense Rank（同値同順位）で計算。

### ハイライト行（重点買い）

`基準オッズ 10〜49倍` かつ `調教スコア ≥ 70` の馬は行全体をハイライト表示。

## 5. 調教スコア計算ロジック

オリジナルスコア（0〜100点）を以下の要素から計算し、S/A/B/C/Dのグレードに変換。

| 要素 | 配点 | 説明 |
|---|---|---|
| A. 追切指数 | 0〜45 | ≥75→45pt / ≥70→35pt / ≥60→20pt / ≥50→5pt |
| B. 仕上指数 | 0〜20 | ≥75→20pt / ≥70→15pt / ≥60→8pt |
| C. 調教矢印 | −10〜+5 | ↑↑→+5 / ↑→+3 / →→0 / ↓→−5 / ↓↓→−10 |
| D. 坂路使用 | +5 | course_saka='01' |
| E. 一週前指数 | +5 | 一週前追切指数≥70 |
| F. 入厩日数 | −3〜+3 | 〜14日→+3 / 〜21日→+1 / 〜28日→0 / 〜35日→−3 |
| G. 放牧先ランク | −5〜+5 | A/B→+5 / C→+2 / D→−3 / E→−5 |
| H. 調教量評価 | −5〜+5 | A→+5 / B→+2 / C→0 / D→−5 |
| I. クラス補正 | ×0.95 | 3勝クラス / OP |

**グレード判定：**
- S：≥80 / A：≥70 / B：≥60 / C：≥40 / D：＜40

## 6. API仕様

### GET /api/races/:raceKey（レース情報）

**パラメータ：** `raceKey` = 8桁キー

**レスポンス：** レース基本情報（T_BAC）

```json
{
  "course_code": "08", "year_code": "26", "kai": "3",
  "day_code": "2", "race_num": "10",
  "ymd": "20260426", "race_name": "センテニアル・ＰＳ",
  "race_name_9char": "センテニアルＰＳ",
  "distance": "1800", "tds_code": "1",
  "grade": "5", "class": "16",
  "heads": "18", "start_time": "1450",
  "migihidari": "1", "naigai": "1"
}
```

### GET /api/races/:raceKey/entries（出走馬一覧）

**レスポンス：** `{ source: "entries", rows: [...] }`

各行は T_KYI LEFT JOIN T_CYB のデータ。主なカラム：

| カラム | 説明 |
|---|---|
| uma_num | 馬番 |
| waku_num | 枠番 |
| uma_name | 馬名 |
| kyakushitsu | 脚質コード |
| kijun_odds | 基準単勝オッズ |
| kijun_ninki | 基準人気 |
| joho_index | 情報指数 |
| idm | IDM |
| goal_juni | 展開（ゴール）順位 |
| kyusha_index | 厩舎指数 |
| kishu_name | 騎手名 |
| trainer_name | 調教師名 |
| ten_index_juni | テン指数順位（前3F） |
| agari_index_juni | 上がり指数順位（後3F） |
| ato3f_juni | 後3F順位 |
| blinker | ブリンカー |
| chokyo_yajirushi | 調教矢印 |
| nyukyu_nichi_mae | 入厩何日前 |
| hohbokusaki_rank | 放牧先ランク |
| joken_class | 条件クラス |
| oi_index | 追切指数（T_CYB） |
| shiage_index | 仕上指数（T_CYB） |
| chokyo_ryo_hyoka | 調教量評価（T_CYB） |
| course_saka | 坂路使用フラグ（T_CYB） |
| isshuumae_oi_index | 一週前追切指数（T_CYB） |
