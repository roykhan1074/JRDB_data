## 2. サマリーカード

| カード | 内容 | 計算方法 |
|---|---|---|
| 総レコード数 | 全テーブルの行数合計 | `Σ total` |
| テーブル数 | 対象テーブルの数 | `data.length`（固定6件） |
| ロード済み日数 | 最もロード日数が多いテーブルの日数 | `MAX(days)` |
| 最新データ日付 | 全テーブルで最も新しい日付 | `MAX(maxDate)` |

---

## 3. テーブル一覧

| 列 | 内容 |
|---|---|
| テーブル | テーブル名（バッジ付き色分け） |
| 説明 | テーブルの日本語説明 |
| 総行数 | `information_schema.tables` から取得（概算） |
| ロード日数 | `COUNT(DISTINCT date_col)` |
| 最古データ | `MIN(date_col)` |
| 最新データ | `MAX(date_col)` |

## 4. API仕様

### GET /api/stats

**レスポンス（JSON配列）**

```json
[
  {
    "table":   "T_BAC",
    "label":   "レース基本情報",
    "prefix":  "BAC",
    "total":   289319,
    "minDate": "20200105",
    "maxDate": "20260428",
    "days":    2286
  }
]
```

**キャッシュ：** 30秒間メモリキャッシュ（重いクエリのため）

---

## 5. 動作

- ページロード時に自動取得（`loadStats()` 即時呼び出し）
- 「🔄 更新」ボタンで再取得
- 取得中はボタン無効化・スピナー表示
- 右下に最終更新日時を表示
