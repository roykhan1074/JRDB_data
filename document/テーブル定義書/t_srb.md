# テーブル定義書: t_srb

## テーブル概要

| 項目 | 内容 |
|------|------|
| テーブル名 | `t_srb` |
| 論理名 | SRB成績速報（JRDB生テーブル） |
| エンジン | InnoDB |
| 文字セット | utf8mb4 |
| ソースファイル | `SRB` + YYMMDD + `.txt` |
| 説明 | JRDBのSRBファイルを固定長からCSV変換しそのまま格納したRAWテーブル。レース単位の速報データ（コーナー通過・コメント等）を保持。 |

## 主キー

`course_code` + `year_code` + `kai` + `day_code` + `race_num`

## 備考

- レース単位のデータ（SED が馬単位なのに対し SRB はレース単位）
- `numeric_data` は固定フォーマットで複数の数値を連結したフィールド
- `corner_order_*` は全出走馬の順位を文字列として格納（例: `1-3,2-5,3-7...`）
